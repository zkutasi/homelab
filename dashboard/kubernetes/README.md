# Kubernetes Dashboard

Manage Kubernetes clusters via a UI, not from the CLI. Observability, notifications, insights.

## Requirements

- Free and Open source, preferably 0$ cost
- Possibility to deploy inside Kubernetes or use on the local machine via kubeconfig files

## Contenders

- [Dashboard](https://github.com/kubernetes-retired/dashboard) - General-purpose web UI for Kubernetes clusters. Archived in 2026.
- [Lens](https://lenshq.io/) - The way the world runs Kubernetes. $$$
  - [OpenLens](https://github.com/MuhammedKalkan/OpenLens) - Fork of Lens after Lens closed its source. Unmaintained as of 2025.
  - [FreeLens](https://freelensapp.github.io/) - Fork of OpenLens
- [K9s](https://k9scli.io/) - Kubernetes CLI To Manage Your Clusters In Style!
- [Headlamp](https://headlamp.dev/) - A Kubernetes web UI that is fully-featured, user-friendly and extensible
- [Aptakube](https://aptakube.com/) - Modern, lightweight and multi-cluster Kubernetes GUI. Available on Windows, macOS and Linux. $$$
- [Kite](https://github.com/kite-org/kite) - A lightweight, modern Kubernetes dashboard that unifies multi-cluster and resource management, enterprise-grade user governance (OAuth, RBAC, and audit logs), and AI agents in one workspace. Not just a tool, but more like a platform.
- [Octant](https://octant.dev/) - Highly extensible platform for developers to better understand the complexity of Kubernetes clusters. From VMWare. Archived in 2023.
- [Devtron](https://devtron.ai/) - The only Kubernetes dashboard you need
- [Skyhook Radar](https://radarhq.io/) - The missing open source Kubernetes UI. Topology, event timeline, and service traffic — plus resource browsing and Helm management.
- [Kubewall](https://github.com/kubewall/kubewall) - Single-Binary Kubernetes Dashboard with Multi-Cluster Management & AI Integration. (OpenAI / Claude 4 / Gemini / DeepSeek / OpenRouter / Ollama / Qwen / LMStudio)
