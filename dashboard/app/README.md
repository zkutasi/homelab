# App Dashboard

A dashboard for all the apps running, having an easy interface to navigate to these apps, check basic info like health and app-specifics alike. Additionally useful features like weather, news, stockmarket, etc... widgetable, extendable.

## Requirements

- Free and Open source, preferably 0$ cost
- Infrastructure as Code support or at least an API
- Widgets, integrations, high extensibility

## Contenders

- [Homarr](https://homarr.dev/) - A modern and easy to use dashboard. 40+ integrations. 10K+ icons built in. Authentication out of the box. No YAML, drag and drop configuration. Also it uses more RAM & CPU than the usual other alternatives.
- [Homer](https://homer-demo.netlify.app/) - Homer is a dead simple static HOMepage for your servER (or anything else) to keep your services and favorite links on hand, based on a simple yaml configuration file.
- [Heimdall](https://heimdall.site/) - An Application dashboard and launcher. Stagnant since 2025, no commits, or PRs.
- [Homepage](https://gethomepage.dev/) - A highly customizable homepage (or startpage / application dashboard) with Docker and service API integrations.
- [Dashy](https://dashy.to/) - A self-hostable personal dashboard built for you. Includes status-checking, widgets, themes, icon packs, a UI editor and tons more!
- [Glance](https://github.com/glanceapp/glance) - A self-hosted dashboard that puts all your feeds in one place
- [TraLa](https://www.trala.fyi/) - A simple, modern, and dynamic dashboard for your Traefik services.
- [DashLit](https://dashlit.cwec.dev/) - DashLit is self-hosted startpage for your server.
- [Dashwise](https://dashwise-next.vercel.app/) - Dashwise, an all-in-one homelab dashboard. Primary focus is on GUI editing.
- [Hestia](https://hestiacp.com/) - A lightweight and powerful control panel for the modern web.
- [LabDash](https://github.com/AnthonyGress/lab-dash) - An open-source dashboard designed to manage your server/homelab applications and services.
- [Mafl](https://mafl.hywax.space/) - Minimalistic flexible homepage. Abandoned since 2024.
