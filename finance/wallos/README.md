## Usage

### Deploy the app

1. Check which version you want to install, or leave empty to take the latest available version

    ```bash
    curl -s https://oci.trueforge.org/v2/truecharts/wallos/tags/list | jq
    ```

2. Install with the provided script

    ```bash
    ./deploy-k8s.sh
    ```
