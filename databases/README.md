# Database tools

Backup handling, queries, traversal, remote connectivity and much more.

## Requirements

- Free and Open source, preferably 0$ cost

## Contenders

- [Adminer](https://www.adminer.org/) - Database management in a single PHP file
- [dbgate](https://www.dbgate.io/) - Database manager for MySQL, PostgreSQL, SQL Server, MongoDB, SQLite and others. Runs under Windows, Linux, Mac or as web application
- [Databasus](https://databasus.com/) - Database backup tool (PostgreSQL, MySQL\MariaDB and MongoDB)
- [Portabase](https://portabase.io/) - Database backup & restore tool for PostgreSQL, MySQL, MariaDB, Firebird SQL, SQLite, MongoDB, Redis and Valkey
