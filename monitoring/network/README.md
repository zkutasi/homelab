# Network scanners & monitors

Applications, that periodically scans the subnet of the network your hosts are deployed and look for certain things: services, open ports, or even metrics. Able to notify also of changes, so you will know when a Host is down or even a completely new host is attached.

## Requirements

- Free and Open source, preferably 0$ cost

## Contenders

- [ArpWatch](https://www.tecmint.com/monitor-ethernet-activity-in-linux/) - CLI Tool
- [LibreNMS](https://www.librenms.org/) - SNMP-based network monitoring and discovery
- [NetAlertX](https://netalertx.com/) - Formerly Pi-Alert, a similar look & feel to Pi-hole
- [WatchYourLAN](https://github.com/aceberg/WatchYourLAN) - Only watches for Hosts, nothing deeper
- [RunZero](https://www.runzero.com/) - Network threat detection and exposure management
- [Portracker](https://github.com/mostafa-wahied/portracker) - Network wide port tracking and diccovery, supports multi-hosts too
- [PortNote](https://github.com/crocofied/PortNote) - Minimalist port-tracker
